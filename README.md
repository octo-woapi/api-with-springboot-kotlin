## Getting started

> **REQUIRED** : You must have Java 11 installed on your computer

Clone the project : 
```shell
git clone https://github.com/octo-woapi/api-with-springboot-kotlin.git
```

Start the project : 
```shell
./gradlew springbootrun
```

Access the API : 
```shell
curl -G http://localhost:8080/products/
```
